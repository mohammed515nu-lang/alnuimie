# 🔧 حل مشكلة "The authorization code may have expired or already been used"

## 🔍 المشكلة

عند محاولة تسجيل الدخول عبر Google، تظهر رسالة:
```
❌ The authorization code may have expired or already been used. Please try logging in again.
```

## 📋 الأسباب المحتملة

### 1. **الكود تم استخدامه مرتين**
- يحدث هذا عادة عندما يتم استدعاء callback مرتين
- قد يحدث بسبب React StrictMode في التطوير
- أو بسبب إعادة تحميل الصفحة أثناء المعالجة

### 2. **الكود انتهت صلاحيته**
- Google authorization codes تنتهي صلاحيتها بسرعة (عادة خلال دقائق)
- إذا استغرقت العملية وقتاً طويلاً، قد ينتهي الكود

### 3. **مشكلة في Redirect URI**
- إذا كان Redirect URI غير متطابق تماماً، قد يفشل التبادل

---

## ✅ الحلول

### الحل 1: المحاولة مرة أخرى (الأسهل)

1. **ارجع** إلى صفحة تسجيل الدخول
2. **اضغط** على زر "تسجيل الدخول عبر Google" مرة أخرى
3. **تأكد** من عدم إعادة تحميل الصفحة أثناء العملية

---

### الحل 2: التحقق من Redirect URI

1. **شغّل**:
   ```bash
   cd server
   npm run check-oauth
   ```

2. **انسخ** Redirect URI الذي يظهر

3. **اذهب** إلى [Google Cloud Console](https://console.cloud.google.com/apis/credentials)

4. **اضغط** على OAuth 2.0 Client ID الخاص بك

5. **تأكد** من أن Redirect URI في **Authorized redirect URIs** مطابق تماماً:
   - يجب أن يكون: `http://localhost:3000/auth/google/callback` (للـ Development)
   - بدون مسافات إضافية
   - بدون `/` في النهاية (إلا إذا كان موجوداً)

6. **احفظ** التغييرات

7. **انتظر** 5-10 دقائق حتى تنتشر التغييرات

---

### الحل 3: التحقق من Environment Variables

1. **تأكد** من أن ملف `server/.env` يحتوي على:
   ```env
   GOOGLE_CLIENT_ID=القيمة_الصحيحة
   GOOGLE_CLIENT_SECRET=القيمة_الصحيحة
   FRONTEND_URL=http://localhost:3000
   ```

2. **تأكد** من:
   - عدم وجود مسافات إضافية
   - عدم وجود علامات اقتباس
   - القيم ليست placeholder values

3. **أعد تشغيل** السيرفر:
   ```bash
   cd server
   npm start
   ```

---

### الحل 4: مسح Cache و Cookies

1. **امسح** cache المتصفح
2. **امسح** cookies للموقع
3. **جرب** مرة أخرى

---

## 🔍 التحقق من الإعداد

شغّل هذا الأمر للتحقق من أن كل شيء مُعد بشكل صحيح:

```bash
cd server
npm run check-oauth
```

يجب أن ترى:
```
✅ Google OAuth جاهز للاستخدام!
```

## ⚠️ ملاحظات مهمة

1. **لا تعيد تحميل الصفحة** أثناء عملية تسجيل الدخول
2. **انتظر** حتى تكتمل العملية
3. **إذا فشلت**، ارجع إلى صفحة تسجيل الدخول وجرب مرة أخرى
4. **تأكد** من أن السيرفر يعمل قبل المحاولة

1. **شغّل** `npm run check-oauth` للتحقق من الإعداد

4. **تأكد** من أن Redirect URI مطابق تماماً في Google Cloud Console
