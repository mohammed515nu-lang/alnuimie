## 1️⃣ صفحة تسجيل الدخول

### المسار:
- URL: `http://localhost:3000/login`
- أو: `http://localhost:3000/` (الصفحة الرئيسية)

### الخطوات:
1. **شغّل** التطبيق
2. **اذهب** إلى صفحة تسجيل الدخول
3. **التقط** صورة للصفحة كاملة
4. **تأكد** من:
   - وضوح النصوص
   - ظهور جميع الحقول
   - ظهور زر "تسجيل الدخول عبر Google" (إن وجد)

### نصائح:
- استخدم **F11** لإخفاء شريط المتصفح
- استخدم **Windows + Shift + S** (Windows) أو **Cmd + Shift + 4** (Mac) للالتقاط
- أو استخدم أدوات مثل **Snipping Tool** أو **Lightshot**

## 2️⃣ لوحة التحكم (Dashboard)

### للعميل:
- URL: `http://localhost:3000/client/profile`
- أو: `http://localhost:3000/client/dashboard`

### للمقاول:
- URL: `http://localhost:3000/contractor`

### الخطوات:
1. **سجل دخول** كعميل أو مقاول
2. **اذهب** إلى لوحة التحكم
3. **التقط** صورة للصفحة
4. **تأكد** من:
   - ظهور الإحصائيات
   - ظهور الرسوم البيانية (إن وجدت)
   - ظهور القائمة الجانبية

### نصائح:
- **التقط** صورة للشاشة كاملة
- **أو** التقط صور متعددة لأقسام مختلفة

## 5️⃣ نموذج الدفع (Stripe)

### المسار:
- يظهر عند إتمام عملية شراء أو دفع

### الخطوات:
1. **ابدأ** عملية دفع
2. **انتظر** ظهور نموذج Stripe
3. **التقط** صورة للنموذج
4. **ملاحظة**: يمكنك استخدام بيانات تجريبية:
   - Card: `4242 4242 4242 4242`
   - Date: أي تاريخ مستقبلي
   - CVC: أي 3 أرقام

## 📝 إضافة الصور في الأطروحة

### في Word:
1. **ضع** المؤشر حيث تريد الصورة
2. **Insert** → **Pictures** → **This Device**
3. **اختر** الصورة
4. **اضبط** الحجم (Width: 15-16 cm)
5. **أضف** عنوان أسفل الصورة:
   - **References** → **Insert Caption**
   - النص: "الشكل X-X: [وصف الصورة]"

### التنسيق:
- **الوسط**: محاذاة الصورة في الوسط
- **التفاف**: None (بدون تفاف)
- **الجودة**: عالية
