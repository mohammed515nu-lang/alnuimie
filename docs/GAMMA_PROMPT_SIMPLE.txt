           برومبت Gamma - نظام إدارة المقاولات
           (انسخ هذا البرومبت بالكامل)
═══════════════════════════════════════════════════════════════

أنشئ عرض تقديمي احترافي باللغة العربية عن "نظام إدارة المقاولات - المستقبل لإدارة المقاولات" لمشروع تخرج في قسم الهندسة المعلوماتية.

المتطلبات:
- 16 شريحة بتصميم احترافي
- ألوان: Primary #3A424F (رمادي داكن)، Secondary #4A5568، Accent #5A6578
- خطوط واضحة للعربية (Cairo أو Arial)
- أضف رسوم بيانية ومخططات تلقائياً حيثما كان ذلك مناسباً
- استخدم أيقونات ورسوم توضيحية احترافية

المحتوى:

الشريحة 1 - صفحة العنوان:
نظام إدارة المقاولات
المستقبل لإدارة المقاولات
Construction Management System
مشروع تخرج - قسم الهندسة المعلوماتية - جامعة الجزيرة الخاصة
العام الدراسي: 2025-2026

الشريحة 2 - نظرة عامة:
نظام شامل لإدارة المشاريع البنائية يربط بين المقاولين والعملاء مع إدارة متكاملة للمواد والمدفوعات. الهدف: تحويل إدارة المقاولات من الطرق التقليدية إلى النظام الرقمي. أضف رسم بياني يوضح الفرق بين الطريقة التقليدية والرقمية.

الشريحة 3 - المشكلة:
التحديات: استخدام طرق تقليدية (أوراق، Excel)، صعوبة تتبع المشاريع والمواد، عدم وجود تواصل فعال، صعوبة إدارة المدفوعات، عدم وجود تقارير دقيقة. أضف مخطط دائري يوضح نسبة المشاكل.

الشريحة 4 - الحل:
نظام إدارة المقاولات الرقمي يوفر: إدارة شاملة (المشاريع، المواد، المدفوعات)، تقارير تفاعلية (مالية، مشاريع، مواد)، واجهة سهلة الاستخدام (بديهية، متجاوبة، حديثة). أضف مخطط مقارنة Before/After.

الشريحة 5 - الميزات الرئيسية:
للمقاول: لوحة تحكم شاملة، إدارة المشاريع، إدارة المواد والموردين، إدارة المدفوعات، التقارير المالية.
للعميل: متابعة المشاريع، إرسال الطلبات، التواصل المباشر، عرض التقارير.
أضف أيقونات لكل ميزة.

الشريحة 6 - التقنيات المستخدمة:
Frontend: React.js 19.2.0, React Router DOM, CSS3
Backend: Node.js, Express.js 4.18.2, MongoDB 8.0.3, Mongoose
الأمان: JWT, bcryptjs, Google OAuth 2.0
النشر: Vercel, Render, MongoDB Atlas
أضف شعارات التقنيات.

الشريحة 7 - البنية المعمارية:
Frontend (React.js) على Vercel → HTTPS/REST API → Backend (Node.js/Express) على Render → MongoDB Connection → Database (MongoDB Atlas). أضف رسم تخطيطي للبنية المعمارية.

الشريحة 8 - قاعدة البيانات:
10 Models رئيسية: User (المستخدمون), Project (المشاريع), Material (المواد), Supplier (الموردين), Purchase (المشتريات), Payment (المدفوعات), Issue (إصدار المواد), Contract (العقود), Request (الطلبات), Report (التقارير). أضف مخطط ERD مبسط يوضح العلاقات.

الشريحة 9 - الواجهات الرئيسية:
صفحات المقاول: لوحة التحكم، إدارة المشاريع، إدارة المواد، إدارة المدفوعات، التقارير.
صفحات العميل: لوحة التحكم، المشاريع، إرسال طلب، التقارير.
أضف أيقونات للصفحات.

الشريحة 10 - الأمان:
تسجيل دخول آمن (كلمات مرور مشفرة، JWT Tokens, Google OAuth)، حماية البيانات (HTTPS، حماية API، التحقق من الصلاحيات)، الأمان على مستوى قاعدة البيانات. أضف رسم تخطيطي يوضح طبقات الأمان.

الشريحة 11 - التكامل مع الخدمات:
Google OAuth: تسجيل دخول سريع وآمن، لا حاجة لكلمة مرور.
Stripe Payment: دفع إلكتروني آمن، دعم البطاقات الائتمانية، فواتير تلقائية.
أضف شعارات Google و Stripe.

الشريحة 12 - النتائج:
نظام متكامل يعمل بالكامل، واجهة مستخدم حديثة ومتجاوبة، قاعدة بيانات منظمة (10 Models)، نظام أمان قوي، نشر على السحابة (3 منصات). أضف مخطط أعمدة يوضح الإنجازات.

الشريحة 13 - الإحصائيات:
10 Models في قاعدة البيانات، 15+ صفحة في الواجهة، 20+ API Endpoint، 100% متجاوب مع جميع الأجهزة، 99.9% وقت تشغيل، نظام أمان متعدد الطبقات، نشر على 3 منصات سحابية. أضف مخطط دائري أو أعمدة للأرقام.

الشريحة 14 - التحديات والحلول:
التحديات: تعقيد إدارة الحالة في React، التكامل مع Stripe Payment، الأمان وحماية البيانات.
الحلول: استخدام React Hooks، وثائق Stripe الشاملة، JWT + bcrypt + Google OAuth.
أضف مخطط مقارنة.

الشريحة 15 - المستقبل:
التحسينات المستقبلية: تطبيق جوال (iOS و Android)، ذكاء اصطناعي (التنبؤ بالتكاليف)، تخطيط المشاريع المتقدم (Gantt Charts)، إدارة الموارد البشرية. أضف مخطط زمني للخطط المستقبلية.

الشريحة 16 - الخلاصة:
نظام شامل ومتكامل لإدارة المقاولات، استخدام تقنيات حديثة ومتطورة، واجهة مستخدم سهلة ومتجاوبة، جاهز للاستخدام الفعلي. النتيجة: نظام متكامل يساهم في التحول الرقمي لقطاع المقاولات. شكراً لكم - Thank You - أسئلة؟

التنسيق:
- العناوين: Bold، 32-36 نقطة
- النصوص: 18-24 نقطة
- Bullet Points واضحة
- ألوان متسقة
- رسوم احترافية

أضف تلقائياً في كل شريحة: رسوم بيانية، مخططات تدفق، مخططات ERD، مخططات مقارنة، أيقونات، رسوم توضيحية حيثما كان ذلك مناسباً ومفيداً.
