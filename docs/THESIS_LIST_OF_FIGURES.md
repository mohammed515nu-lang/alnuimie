## ملاحظات للتنسيق في Word:

### كيفية إنشاء قائمة الأشكال التلقائية:

1. **أضف تسميات للأشكال:**
   - حدد الشكل
   - References → Insert Caption
   - Label: اختر "Figure" أو "شكل"
   - Caption: اكتب عنوان الشكل
   - Position: Below selected item (أسفل العنصر المحدد)

2. **أنشئ قائمة الأشكال:**
   - References → Insert Table of Figures
   - Caption label: اختر "Figure" أو "شكل"
   - Formats: اختر التنسيق المطلوب
   - اضغط OK

3. **تحديث القائمة:**
   - عند إضافة أشكال جديدة، اضغط بزر الماوس الأيمن على القائمة
   - اختر "Update Field"
   - اختر "Update entire table"

## التنسيق المطلوب:

### في Word:
- **العنوان**: "قائمة الأشكال" (في الوسط، Bold، 16 نقطة)
- **الترقيم**: الشكل X-Y (X = رقم الفصل، Y = رقم الشكل)
- **المحاذاة**: 
  - رقم الشكل: يسار
  - العنوان: يسار
  - رقم الصفحة: يمين
- **المسافات**: متساوية بين الصفوف
- **الخط**: 12-14 نقطة

### مثال التنسيق:
```
قائمة الأشكال

الشكل 4-1    التصميم المعماري العام للنظام (Client-Server) ................... 25
الشكل 4-2    نموذج مخطط ERD مبسط يوضح العلاقات بين الكيانات ............... 26
```
