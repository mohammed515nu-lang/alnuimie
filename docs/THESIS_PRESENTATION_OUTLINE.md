# 📊 عرض بوربوينت - نظام إدارة المقاولات

## ملاحظة حول "تصميم" في قائمة الأشكال

**"تصميم"** يعني **Mockups** أو **Wireframes** - أي:
- التصاميم الأولية للواجهات قبل التنفيذ
- الرسوم التخطيطية (Sketches) التي توضح كيف ستبدو الواجهات
- النماذج الأولية (Prototypes) للواجهات

**مثال:**
- **الشكل 4-3: تصميم لوحة تحكم المقاول** = رسم تخطيطي يوضح كيف ستبدو لوحة التحكم قبل البرمجة
- يمكن أن يكون: رسم يدوي، أو تصميم في Figma/Adobe XD، أو Wireframe

## ملاحظات للعرض:

### التصميم:
- استخدم ألوان متسقة (Primary: #3A424F)
- خط واضح (Cairo للعربية)
- صور عالية الجودة
- رسوم بيانية واضحة

- كل شريحة: 3-5 نقاط رئيسية

- أضف رسوم بيانية
- تجنب النصوص الطويلة

### الوقت:
- العرض الكامل: 15-20 دقيقة
- كل شريحة: 1-2 دقيقة
- وقت للأسئلة: 5-10 دقائق
